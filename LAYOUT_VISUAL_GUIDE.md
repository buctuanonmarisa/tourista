## Component Breakdown

### Top Navigation Bar

```
┌─────────────────────────────────────────────────────────────┐
│ 🗺️ Logo    [Search Bar]    🔔 📊 ➕ 👤                    │
└─────────────────────────────────────────────────────────────┘
  ↑           ↑                ↑  ↑  ↑  ↑
  Logo        Search           Notifications
              (centered)       Dashboard
                               Post Vlog
                               Profile
```

**Features**:
- Logo: Clickable, returns to browse
- Search: Centered, with clear button
- Actions: Right-aligned buttons
- Sticky: Stays at top while scrolling

## Interaction Flow

### Browsing Vlogs

```
1. User opens Tourista
   ↓
2. Browse page loads with grid of vlog cards
   ↓
3. User can:
   a) Search for vlogs
   b) Filter by Vibe, Region, or Budget
   c) Click on a vlog card
   ↓
4. Detail panel opens (right side on desktop, overlay on mobile)
   ↓
5. User can:
   a) View vlog details
   b) Like/comment
   c) Unlock premium content
   d) Navigate to next/previous vlog
   e) Close detail panel
```

### Filtering Vlogs

```
1. User clicks on filter tab (Vibe, Region, or Budget)
   ↓
2. Filter chips appear below tab
   ↓
3. User clicks on a chip to select filter value
   ↓
4. Vlog grid updates to show filtered results
   ↓
5. User can click on another tab to change filter category
   ↓
6. Previous filter value is remembered
```

## Color & Typography

### Colors
- **Primary Green**: #2A7A50 (buttons, active states)
- **Light Green**: #E6F5ED (hover backgrounds)
- **Text Primary**: #162A1F (main text)
- **Text Secondary**: #567268 (labels, metadata)
- **Border**: rgba(27,92,58,.13) (subtle dividers)
- **Blue**: #1a73e8 (selection, primary actions)
- **Yellow**: #D08A0A (credits, unlock)

### Typography
- **Logo**: 20px, bold, green
- **Titles**: 18px, bold, dark
- **Labels**: 11px, uppercase, secondary
- **Body**: 14px, regular, primary
- **Metadata**: 13px, regular, secondary
