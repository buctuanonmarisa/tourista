## 🗄️ Database Structure

### **Vlog Model**
Each clip is stored as a separate vlog with:
- `title`: "{Destination}: {Clip Title}"
- `location`: "{Spot}, {Country}"
- `country`: Country name
- `region`: Geographic region
- `vibe`: Travel style category
- `youtubeUrl`: YouTube Shorts URL
- `coverImage`: Auto-generated YouTube thumbnail
- `authorId`: Associated vlogger
- `views`: Random 1000-6000
- `likes`: Random 100-600
- `trending`: First 2 clips per destination

### **ItineraryDay Model**
Each vlog has one itinerary day with:
- `day`: Sequential number (1-5)
- `activity`: Clip title
- `description`: Clip description
- `mediaUrl`: YouTube Shorts URL
- `mediaType`: "video"
- `clipDescription`: JSON with media array
- `highlights`: What makes this clip special
- `tips`: Viewing tips

### **API Endpoint**
`/api/tourme/clips?spot={spot}&country={country}`

**How it works:**
1. Searches for vlogs matching spot and country
2. Extracts video clips from itinerary days
3. Returns up to 8 clips sorted by trending/views/likes
4. Includes vlogger info, thumbnails, and metadata

**Response Format:**
```json
{
  "clips": [
    {
      "id": "clip-id",
      "url": "https://www.youtube.com/shorts/...",
      "title": "Clip title",
      "description": "Clip description",
      "day": 1,
      "vlogId": "vlog-id",
      "vlogTitle": "Full vlog title",
      "location": "El Nido, Philippines",
      "country": "Philippines",
      "coverImage": "thumbnail-url",
      "author": {
        "id": "user-id",
        "handle": "MarisolRoams",
        "initials": "MS",
        "avatarColor": "ag",
        "verified": true
      },
      "views": 3456,
      "likes": 234
    }
  ]
}
```

## 🎯 How It Works in Tour Me

### **User Flow:**

1. **Open Tour Me** → See world map with 10 destination pins
2. **Click a Pin** (e.g., El Nido) → Map switches to place view
3. **See "Things to do"** → 5 YouTube Shorts clips appear
4. **Click a Clip** → Video plays inline with autoplay
5. **Video Ends** → Automatically advances to next clip
6. **Loop** → After last clip, returns to first clip

### **Clip Display:**

**Inactive Clip (Thumbnail):**
- Clip number badge
- YouTube thumbnail image
- Play button overlay
- Title and description
- Vlogger attribution

**Active Clip (Playing):**
- Full video player (9:16 aspect ratio)
- YouTube embed with autoplay
- Header with clip number and title
- Description and vlogger info below
- Auto-advances on video end

## 🚀 Running the Seed Script

### **First Time Setup:**
```bash

```

### **Re-run (Updates Existing):**
```bash
npm run db:seed:tourme-real
```

The script uses `upsert` so it's safe to run multiple times. It will:
- Update existing vlogs with new data
- Create new vlogs if they don't exist
- Update existing itinerary days
- Create new itinerary days if needed
